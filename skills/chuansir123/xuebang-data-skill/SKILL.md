# 学邦 EDU.BOSS 今日数据查看

## 目标
登录 `https://boss.xuebangsoft.net/eduboss/`，读取首页“今日数据预览”和待办区块，并整理成简洁汇总返回。

## 凭据管理
- 这是一个可发布技能，`SKILL.md` 中不得包含任何真实账号、密码、Cookie、Token 或其他敏感信息。
- 首次使用时，如果本地未保存学邦凭据，先向用户索取：
  - 登录账号
  - 登录密码
- 获取后，将凭据写入本地私有笔记 `workspace/TOOLS.md`。
- 后续执行时，优先从 `workspace/TOOLS.md` 读取凭据；只有在缺失或用户要求更新时才再次询问。
- 如果用户明确要求“不保存”，则仅本次使用，任务结束后不在任何文件中落盘。

## 执行时机
当用户表达以下意图时直接使用本技能，不需要用户显式说“使用技能”：
- 查看学邦今日数据
- 查看学邦今日经营数据
- 看下学邦今天数据
- 查学邦待办
- 查 EDU.BOSS 今日概览

## 站点信息
- 登录页：`https://boss.xuebangsoft.net/eduboss/`

## 执行步骤
1. 先检查 `workspace/TOOLS.md` 是否已有“学邦 EDU.BOSS”相关凭据。
2. 如果没有凭据，先向用户索取账号密码，再征求是否保存到本地私有笔记。
3. 如果用户同意保存，将凭据写入 `workspace/TOOLS.md` 的私有条目中。
4. 使用 `browser` 启动默认隔离浏览器，不要直接 `open` 外网地址。
5. 导航到登录页：`https://boss.xuebangsoft.net/eduboss/`
6. 优先通过页面交互填写账号和密码；如常规输入失败，再使用页面脚本直接设置 `input.value`。
7. 点击登录按钮。
8. 等待跳转到首页并加载完成。
9. 读取首页关键指标，优先提取这些字段：
   - 新增客户
   - 新增合同
   - 收款金额
   - 课消金额
10. 读取待办或提醒类字段，优先提取这些字段：
   - 待分配客户
   - 待跟进客户
   - 预约上门
   - 待试听
   - 待回访学员
   - 待考勤
   - 课堂报告
11. 若页面存在 `iframe`，继续进入包含首页数据的 frame 内获取内容。

## 输出要求
按下面结构返回，缺失字段写“未读取到”，不要编造：

```
## 📊 今日数据预览（YYYY-MM-DD）

| 指标 | 今日 | 昨日 |
|------|------|------|
| 🧑 新增客户 | **X** | Y |
| ...

## 📋 待处理事项
| 事项 | 数量 |
|------|------|
| ...

## 📌 小结
...
```

## 约束
- 不要因为个别字段缺失就中止，先返回已拿到的数据。
- 如果登录失败，明确说明失败步骤和页面提示。
- 如果浏览器工具异常，说明是工具不可用，不要伪造结果。
- 不要在响应中复述完整密码。
