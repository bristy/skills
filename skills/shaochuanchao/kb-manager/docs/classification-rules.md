# Classification Rules

Classify by purpose, not by source website.

- `01_reference`: official docs, APIs, specs, product docs
- `02_learning`: tutorials, articles, learning notes
- `03_projects/<project-name>`: project discussions, plans, meeting notes, design docs
- `04_research`: research, comparisons, competitor analysis, investigations
- `05_playbooks`: SOPs, procedures, repeatable workflows
- `06_prompts`: prompts, reusable instructions, templates
- `00_inbox`: uncertain items that should not be force-classified
