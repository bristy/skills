# 双色球查询技能 v1.2.0

当用户输入"双色球"关键词时，自动获取最新开奖数据，显示查询日期前近 10 期开奖结果、走势分析及下期预测推荐。

## 触发条件

- 关键词：双色球、双色球开奖、双色球结果、双色球查询

## 执行方式

```bash
python C:\Users\Administrator\.openclaw\workspace\skills\double-color-ball\scripts\lottery.py
```

## 功能

1. **开奖号码查询**：动态获取最新开奖结果（每次查询显示截至当天的近 10 期）
2. **走势分析**：红球三区分布、连号、重号、热号统计
3. **预测推荐**：娱乐性质推荐 2 注下期号码（热号为主 + 冷热搭配）

## 数据来源

- 优先从福彩官网 API 获取最新数据（https://www.cwl.gov.cn）
- 网络失败时使用内置缓存数据（定期更新）

## 输出内容

- 最新 10 期开奖结果（期号、日期、红球、蓝球）
- 走势分析（三区分布、连号、重号、热号 TOP5）
- 2 注预测推荐（仅供娱乐）
