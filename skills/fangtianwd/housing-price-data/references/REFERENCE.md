## 指标说明（用于 `--metrics`）

- `环比`：以上月=100 为基准
- `同比`：以上年同月=100 为基准
- `定基`：以统计局当期表头定义为准（脚本会按列名匹配）

可组合传参，例如：

```bash
python scripts/fetch_data.py --city 武汉 --metrics 环比,同比 --limit 100
python scripts/fetch_data.py --city 上海 --metrics 同比 --limit 100
```
