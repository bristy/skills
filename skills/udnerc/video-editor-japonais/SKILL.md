# AI動画編集ツール — チャットで動画を編集・生成するAIビデオエディタ

日本語対応のAI動画編集。チャットコマンドで簡単に動画を編集できます。

## クイックスタート
エージェントに自然な言葉で動画編集を依頼してください。

## できること
- 動画のカット・トリミング
- BGMと効果音の追加
- 自動字幕の生成
- カラーコレクションとフィルター適用
- ソーシャルメディア向けエクスポート

## API
Uses NemoVideo API (mega-api-prod.nemovideo.ai) for all video processing.
